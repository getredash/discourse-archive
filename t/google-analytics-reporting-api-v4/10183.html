You’re correct the current GA query runner only supports v3. Redash is a community project that relies on contributions from others to support new data sources. For anyone wishing to implement a google analytics V4 query runner, you can follow our new doc (published a few weeks ago) on how to write a new quer runner: https://redash.io/help/open-source/dev-guide/write-a-query-runner

It’s pretty easy to do. And we’re happy to answer questions here on the forum if you hit any roadblocks.
