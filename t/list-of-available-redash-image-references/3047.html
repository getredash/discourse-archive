List of available Redash image references?

Support Self Hosted Redash Support
scott January 18, 2019, 11:21pm 1

Issue Summary

https://redash.io/help/open-source/admin-guide/how-to-upgrade says “Update /opt/redash/docker-compose.yml Redash image reference to the one you want to upgrade to.” Where can I see the available images? E.g. I’d like to try the 6.0.0 release, but when I try redash/redash:6.0.0 as the image and run “docker-compose up -d” it says “ERROR: manifest for redash/redash:6.0.0 not found.” My current image is redash/redash:5.0.2.b548. Docker 'latest' image tag won't trigger CloudFormation deployment has a screenshot of something suggesting 6.0.0.b8537 might be a valid image version, but that throws the same error. Where can I get a list of the currently available images?

Technical details:

  • Redash Version: 5.0.2+b5486
  • How did you install Redash: EC2 AMI
scott January 18, 2019, 11:24pm 2

To answer my own question after further poking around, https://hub.docker.com/r/redash/redash/tags has a list of the available images, and 6.0.0.b8537 is indeed one of them. I must have just typed it wrong the first time.
