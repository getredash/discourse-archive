Populate Query Parameter Dropdown List with a query Feature Requests

In many cases, these are values that existing in the database that we’re querying, so populating this list with another query would be the easiest way to ensure that this list is complete/accurate. From a UI perspective, this is a feature that is available for dashboards in Splunk.
