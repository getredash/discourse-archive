justinclift June 28, 2019, 2:48pm 2

In theory (!) the dashboard level filters will appear after the Query Filters have been created.

So, 1st step, create Query Filters:

https://redash.io/help/user-guide/querying/query-filters

Then create the Dashboard Level Filters:

https://redash.io/help/user-guide/dashboards/dashboard-editing/

If that’s not working… it might be bug.

On a related topic, there are Query Parameters too:

https://redash.io/help/user-guide/querying/query-parameters

They’re similar to Query Filters, but are applied at a different stage in the process so suit different cases to the Query Filters.

jesse December 12, 2019, 11:12pm 4

Hi @dribble,

Dashboard level filters have been supported for several versions now. When you enter dashboard edit mode there is a checkbox on the upper-left that lets you “enable dashboard level filters”. So long as your filters have the same string literal name then they will be linked together. It’s not as flexible or convenient as query parameters (and isn’t meant to be). But it’s there nonetheless
