First up, you probably shouldn’t post your database password and secret key on a public web forum

Second: Redash doesn’t use an environment variable called POSTGRES_PASSWORD. Your REDASH_DATABASE_URL should be postgresql://postgres@postgres/postgres. Which looks silly but makes sense when broken down:
