This feature is known as “White Labeling” and it isn’t supported.

Remove white label from Dashboard Feature Requests
White labeling isn’t possible yet because it depends on features that are still in development, like the react migration and allowing parameters on public dashboards. Without these, neutral branding won’t be that useful. The primary use case we’ve seen for white labeling is multi-tenant dashboards.
