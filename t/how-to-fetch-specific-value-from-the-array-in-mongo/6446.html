jesse June 12, 2020, 7:10pm 2

You can do this using the fields key. Assume your collection is called foo, the query looks like this:

{
	"collection": "foo",
	"fields": {
		"_id": 1,
		"type": 2,
		"course_name.0": 3
	}
}
