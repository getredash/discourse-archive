データソースMongoDBのクエリで質問です。

ネストされたデータ（roles）をfieldsで表示しようとしたのですが、以下のクエリでは表示されません。

{
    "collection": "users",
    "fields" : {
        "_id" : 1,
        "email" : 1,
        "roles": 1
    }
}

以下のようにprojectで表示はされましたが、カラムが自動でroles.KEY名となります。

{
    "collection": "users",
    "aggregate": [
        {
            "$project": {
                "_id": "$_id",
                "email": "$email",
                "roles": "$roles"
            }
        }
    ]
}
