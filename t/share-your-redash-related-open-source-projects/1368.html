Here’s a few I found on GitHub:

  • Mattermost slash command: https://github.com/kaakaa/matter-redash
  • CLI to control Redash queries: https://github.com/ariarijp/redashman
  • Javascript API client: https://github.com/teppeis/redash-client
  • R API client: https://github.com/yutannihilation/Redashr
  • Rancher template: https://github.com/yassan/rancher-catalog
  • Redash K8S setup: https://github.com/nanit/redash-kubernetes
  • API proxy for iTunes Connect’s App Analytics, compatible with the url data source: https://github.com/yamitzky/itunes-connect-analytics-api
  • PHP API client: https://github.com/ariarijp/redash-api-client
  • Node.js utility that fetches Redash query results via API and upserts the data to Salesforce objects: https://github.com/codygordon/redash-to-salesforce
  • Android app: https://github.com/Bambina/RedashClient
  • A tool for executing dynamic query of Redash: https://github.com/toritori0318/redash-dynamic-query
  • Python API client: https://github.com/MasamichiIdeue/redquery
  • Convert Redash Query Result to Pandas DataFrame: https://github.com/ariarijp/pandash
  • Create Redash queries based from definition files: https://github.com/kirikiriyamama/redash-query-importer
  • DataDog Agent Redash collector: https://github.com/junjizhi/redash-datadog
  • Insturctions on setting up Redash on GKE: https://github.com/donaldrauscher/redash-gke
  • Ruby API Client: https://github.com/tomoya55/redash-ruby
  • Send Redash query results as CSV attachments: https://github.com/MoveOnOrg/redash-emailer
  • Go API Client: https://github.com/ynishi/redash

An SDK for the programmatic management of Redash, in Go. - GitHub - RecoLabs/redash-go-sdk: An SDK for the programmatic management of Redash, in Go.

An SDK for the programmatic management of Redash. The main component of the SDK is a client , which is a go wrapper of Redash’s REST API.

How we use it: We needed a way to programmatically control our self-hosted Redash instance from our backend services - we developed a custom dashboard screen in our product and Redash solved many issues for us, so we decided to use it as both the backend and the Data Analyst query development IDE.
