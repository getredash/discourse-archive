Are there instructions on how to upgrade to a RC release? I used the AWS AMI for our setup.

arikfr September 16, 2016, 8:21pm 4

It’s the same as regular upgrade (see docs), but you run the fab command with deploy_latest_release:stable=false instead of deploy_latest_release.

Make sure to make a backup of the metadata database before upgrading.
