jesse December 13, 2019, 10:01pm 2

I agree this could be useful to have a Tab Separated Value (TSV) export.

A workaround if you’re okay with scripting it yourself is to use the Redash API to pull the query result as JSON and then output to your desired format. In Python this is as easy as instantiating a new csvWriter with a tab delimiter instead of ,.
