Sort schema columns alphabetically

opened 04:47PM - 27 Jan 20 UTC
susodapop
+65 -6

For us, the column order is generally pretty sensible. We have lots of tables have grouped dimensions vs. metrics or IDs next to friendly names. This one-size-fits-all alphabetical approach is really unfortunate

arikfr July 19, 2020, 12:21pm 2

The reason was that in most cases the order that returns by default is random (based on the time column were added to the table) and alphabetical sort made more sense as it’s easier to reason about.
