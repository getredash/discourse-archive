jesse October 2, 2019, 1:57pm 2

Welcome to the forum. Just add the $toDate aggregation to your pipeline. Like this:

{
  "collection": "orders",
  "aggregate": [
    {
      "$project": {
        "date": "$ts_create"
      },
      "$toDate": "date"
    }
  ],
  "fields": {
    "custom_order_id": 1,
    "date": 2,
    "total_amount": 3
  }
}
