I’m fairly sure Redash uses Python logging, so “WARNING” is not a valid log level, you need to use “WARN”.

However, this will still output INFO messages as this is the standard logging behavior, i.e. all log messages at the specified level OR ABOVE will be captured.

See this tutorial for more details https://www.toptal.com/python/in-depth-python-logging
